# ceil round floor函数
1. C 语言取整函数对比笔记在使用这些函数之前，请确保包含了数学库头文件：C#include <math.h>
2. 函数定义与逻辑
  - | 函数名 | 逻辑 | 2.1 | 2.5 | 2.9 | -2.1 | -2.9 |
| :--- | :--- | :---: | :---: | :---: | :---: | :---: |
| **`ceil`** | 向上取整 | `3.0` | `3.0` | `3.0` | `-2.0` | `-2.0` |
| **`floor`** | 向下取整 | `2.0` | `2.0` | `2.0` | `-3.0` | `-3.0` |
| **`round`** | 四舍五入 | `2.0` | `3.0` | `3.0` | `-2.0` | `-3.0` |
3. 代码演示
~~~
#include <stdio.h>
#include <math.h>

int main() {
    double num = 2.3;

    printf("原始数值: %.1f\n", num);
    printf("向上取整 (ceil):  %.1f\n", ceil(num));  // 结果 3.0
    printf("向下取整 (floor): %.1f\n", floor(num)); // 结果 2.0
    printf("四舍五入 (round): %.1f\n", round(num)); // 结果 2.0

    return 0;
}
~~~
4. 重要提示返回值类型：这三个函数的返回值都是 double。如果你需要存入 int 变量，建议使用强制类型转换：int result = (int)ceil(20000.0 / v);
